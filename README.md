# Faiza's Personal Portfolio 🚀

Welcome to the repository for my personal portfolio website! This project is currently **under construction**. I am designing and building this space from scratch to showcase my work, share my journey, and connect with other professionals.

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **JavaScript**
* **Tailwind CSS**

## 🗺️ Project Roadmap & Status

I am building this site iteratively. Here is the current progress of the project:

### ✅ Completed
* **Homepage:** A clean, welcoming landing page that introduces who I am and sets the tone for the website.
* **About Me Page:** A deeper dive into my background, my education, and the technical and creative skills I bring to the table.
* **Portfolio Page:** A dedicated gallery to showcase my software development projects and design work with links to live demos and repositories.
* **Blog Page:** A space where I plan to share technical write-ups, project insights, and personal experiences in tech and design.
* **Contact Me Page:** A functional contact form and links to my social profiles to make it easy for visitors to reach out.

## 💻 Local Setup

If you want to run this project locally to see the current progress:
1. Clone the repository: `git clone https://github.com/faiza/Portfolio-Website
.git`
2. Open the project folder in your favorite code editor (like VS Code).
3. Open `index.html` in your browser, or use an extension like Live Server to view it dynamically.
